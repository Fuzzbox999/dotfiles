Basics

  • [Mod4] + Return = open a terminal
  • [Mod4] + d = dmenu launcher
  • [Mod4]+[Shift] + a = close window
  • [Mod4]+[Shift] + e = exit dwm

Focus, move and resize

  • [Mod4] + j = focus previous window in the stack
  • [Mod4] + k = focus next window in the stack
  • [Mod4]+[Shift] + Return = switch with master
  • [Mod4] + h or l = resize window
  • [Mod4]+[Shift] + j = move window up the stack
  • [Mod4]+[Shift] + k = move window down the stack
  • [Mod4] + i = increase nb of windows in master area
  • [Mod4] + o = decrease nb of windows in master area

Layouts

  • [Mod4] + $key = select layout (with $key = t, f, g, s or m)
  • [Mod4] + space = toggle between current and previous layout

  • [Mod4]+[Shift] + Space = toggle floating
  • [Mod4]+[Shift] + f = toggle fullscreen
  • [Mod4]+[Shift] + y = toggle client fake fullscreen

Tags

  • [Mod4] + [1 to 9] = select [1 to 9] tag
  • [Mod4] + 0 = display windows of all tags
  • [Mod4]+[Ctrl] + Right or Left = move to next/previous tag
  • [Mod4] + Tab = cycle between current and last tags
  • [Mod4]+[Shift] + $tag = send window to $tag
  • [Mod4]+[Ctrl] + $tag = view all windows of $tag in current tag
  • [Mod4]+[Ctrl]+[Shift] + $tag = add current window to $tag

Gaps

  • [Mod4]+[Alt] + g = toggle gaps
  • [Mod4]+[Alt]+[Shift] + g = reset gaps
  • [Mod4]+[Alt] + i or o = increase inside or outside gaps
  • [Mod4]+[Alt]+[Shift] + i or o = decrease inside or outside gaps

Monitors

  • [Mod4] + Left or Right = focus next or previous monitor
  • [Mod4]+[Shift] + Left ot Right = send window to next or previous monitor
