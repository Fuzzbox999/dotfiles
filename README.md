# dotfiles

Arch Linux on [Thinkpads](https://www.reddit.com/r/thinkpad/) (X220 and T430), with [i3gaps](https://github.com/Airblader/i3) as a wm, a patched [dmenu](https://tools.suckless.org/dmenu/) as an app launcher, and [Pywal](https://github.com/dylanaraps/pywal) along with [wpgtk](https://github.com/BigfootN/wpgtk) as a theme manager.

![Screenie](https://raw.githubusercontent.com/Fuzzbox999/dotfiles/master/Images/Screenshots/screenie.jpg)

System configuration, software and hardware are intricately linked.
Don't just copy the files because they probably won't work on your system.
Read the files, understand what they do, and do your own stuff.
